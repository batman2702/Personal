
## Running the Mobile App

1.  Start the Next.js server: `npm run dev`
2.  Navigate to `mobile/`: `cd mobile`
3.  Install dependencies: `npm install`
4.  Start Expo: `npx expo start`

### Troubleshooting
-   **Android SDK Issue**: If you see "Failed to resolve the Android SDK path", either install Android Studio and set `ANDROID_HOME` or use the **Expo Go** app on your physical Android device.
-   **iOS Simulator**: Press `i` to run on iOS Simulator (requires Xcode).
